# 大湾区 · 地形—制度叠加底图

沿用深圳底图那套设计系统（羊皮纸 / 古地图美学、同一套配色与字体）。把《被水统一、被史三分的海湾》一篇的核心论点，落到一张可平移缩放的分层底稿上。

## 八个图层

1. **山框与主峰** — 罗浮山—九连、莲花山脉、天露—云雾、南岭余脉、西樵孤峰：框定平原的边界。
2. **八门·三江·深槽** — 西/北/东三江挤入一湾；东四门（虎门·蕉门·洪奇沥·横门）注伶仃洋、西四门（磨刀门·鸡啼门·虎跳门·崖门）泄黄茅海；伶仃/矾石两条深槽。深水在东、泥沙在西。
3. **拉链齿序（桥隧）** — 虎门→南沙→港珠澳→深中→黄茅海→狮子洋（在建）→深珠（规划），每颗齿重排一次地价地图；港珠澳 vs 深中的制度对照实验。
4. **三极四膜** — 广佛（无膜）·深港（膜最厚）·珠澳（手术最大）；前海·横琴·南沙·河套四张膜各试一种渗透率。
5. **五港五场** — 深/广全球前六、香港 2024 滑出前十、高栏西岸唯一深水港；五大机场与空域瓶颈。
6. **代谢管线** — 东深供水、珠三角水资源配置（西江水入深）、大藤峡压咸、西电东送、大装置进山：功能湾区画在半个中国上。
7. **轨道环** — 环湾一小时圈示意，广佛线/广深港高铁/城际/深江铁路/澳门轻轨等关键连接。
8. **城市与水名** — 标注层。

## 六个预设视角

全览 / 底盘·三江八门 / 三角形成·一口两殖 / 闭合·拉链南拉 / 三极四膜 / 五港五场 / 代谢管线——每个视角配一句话结论式导读，并自动切换图层与聚焦。

交互：左栏勾选图层；点击城市（含港澳）看定位·产业·风险；悬停口门、桥齿、平台、港口机场、管线见断代与力学注记；滚轮/双指缩放、拖拽平移、双击复位。

## 部署（GitHub Pages）

1. 把本文件重命名为 `index.html` 放到仓库根目录并 push（当前文件名为 `gba-index.html`，仅为与深圳版区分）。
2. Settings → Pages → Source 选 `Deploy from a branch`，Branch 选 `main` / root，保存。
3. 约一分钟后访问 `https://<用户名>.github.io/<仓库名>/`。

本地预览：`python -m http.server` 后打开 `http://localhost:8000`，或直接双击文件。单文件、零外链，墙内可直开。

## 数据与精度

- 行政边界：DataV 公开数据镜像（GeoMapData_CN），道格拉斯—普克简化；香港、澳门及广东其余各市、邻省（江西·广西）仅作背景示意，后者虚绘。
- 八门位置、山框、桥隧齿序、平台范围、港口机场、代谢管线走向均为**示意精度**，非测绘成果；跨海通道按拓扑连接绘制的象征性弧线，非精确线位。
- 经济总量与港口排名为近年公开量级值，非最新精确统计。
