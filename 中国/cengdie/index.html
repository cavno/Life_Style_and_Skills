方法与注记

归类依据：全国 330 余个地级单位按《中国语言地图集》（第 2 版）大区划分逐一归类（169 条显式条目 + 各省默认规则）。过渡地带从主；徐州、信阳、汉中、九江、上饶、镇江等争议或有趣的点位，点击地图可见专门注释。本图为文化示意，非学术精确边界。

「犬牙交错」：与「山川形便」相对的划界原则，自元代行省制度系统化，明清沿袭。其对象不止文化连续体，更是地形完整、文化同质、资源自足的「自然防御单元」——故汉中离蜀、徐州离豫。

结构反铸主体：区划是「人为设计」与「历史路径依赖」的叠加。同时，省界在七十余年运行中反过来生成了新的认同——推普以省级电视台为中介、菜系按省命名、婚姻圈沿省界收敛。「四川人」「广东人」已是真实的次级身份。

数据：行政边界来自阿里云 DataV.GeoAtlas（运行时加载，jsDelivr / GitHub 备援），仅用于可视化。字体：ZCOOL XiaoWei / Noto Serif SC / EB Garamond。
