点选任一辖区

地图上点击行政区、前海★或深汕框，可查看一句话定位、要点与风险；悬停口岸、山峰、箭头可见断代注记。滚轮/双指缩放，拖拽平移，双击复位。

方法注记 — 行政边界据公开行政区划数据（DataV 镜像）经道格拉斯—普克简化（容差约 60 m）；香港、东莞、惠州、中山轮廓为粗略示意；山脊、河道、分水岭、二线与通道走向均为示意精度，非测绘成果。大鹏为功能区，行政隶属龙岗，此处按空间事实单列。

断代注记 — 口岸、港口、通道年份见图中标注与悬停说明；特区管理线 1985 启用、2010 特区扩容至全市、2018 正式撤线。底稿：《深圳：资本主义飞地作为改革之春》综述。v1.0 · 2026-07 · 单文件，可直接部署 GitHub Pages（仓库根目录放置 index.html → Settings › Pages）。
