# 资产负债表重组 · 置换(微信小程序)

把家庭当成一家"无限责任公司",用**资产负债表**的视角重构了那张"房价 → 月供 → 所需收入 → 留存现金"的决策表。整张盘面被翻译成公司金融语言:去杠杆 + 资质重置、用长债锁定核心资产、固收腿覆盖底层负债、看跌期权隔离尾部、其余流动性博取 α。**纯本地计算,零网络请求**——不需要服务器、不需要配置域名、不上传任何数据。

---

## 一、两个页签

### 资产负债表(主视角)
拖动**房产总价**与**妻子税后月收入**两个主滑杆,下面所有部件实时重算:

1. **判定带** — 一句话给出"结构是否成立":结构稳健 / 临界 / 依赖 α / 偿付越线 / 股本不足。
2. **家庭资产负债表(签名元素)** — 左右两栏等高柱,直观呈现恒等式 `资产 = 负债 + 净资产`。
   - 资产:核心资产(房)、境内可投、**境外权益(斜纹标注,受结汇限制)**、安全垫
   - 负债+权益:公积金贷款、商贷、净资产(权益是使两栏配平的"塞子")
3. **核心比率** — 加权债务成本(WACC,仅债务侧)、**DSCR 偿债保障倍数**、全口径覆盖倍数、双层杠杆(房产层 5× / 家庭层 ~1.5×)。
4. **息差 · 两副面孔** — 诚实拆分:**锁定息差**(境内固收 r* − 债务成本,可套利)vs **权益风险溢价**(美股 E[R] − 债务成本,*不是*套利);附保留/砍掉商贷的净现值(NPV)判定与 r* 滑杆。
5. **久期匹配** — 资产久期 vs 负债久期的对照条;一键切到"经营贷"即可看到期限错配如何点亮红色警告(这正是此前否决经营贷置换的原因)。
6. **尾部对冲** — 把安全垫看作一份**看跌期权**:保护深度(可覆盖月数)+ 年化"保费"(为换取确定性放弃的息差)。
7. **人力资本组合** — 妻子=类固收(债券票息,负责下限)/ 本人=权益类(高波动 α,不托底);并给出"固收腿是否独立托起固定支出"的结论。
8. **参数(开账口径)** — 折叠区,全部可改。

### 贷款测算(引擎室)
组合贷计算:总价/首付/公积金额度/双利率/年限,等额本息 ↔ 等额本金切换,含首月/递减/末月与两方式总利息对比。

## 二、与原文档口径的两处校正(重要)

附件那套"家庭即公司"的框架很贴切,但有两处被原文一笔带过、而本工具按此前已核实的分析做了校正:

1. **公积金额度按已核实的 209 万**(110 万家庭基数 ×(1+首套40%+二孩50%)),而非文档随口的"拉满 260 万"。260 万这档仍保留为预设,但明确标注"待核实",需 12329 确认后再切换。
2. **美股 25–40% 的收益是风险溢价,不是锁定息差。** 它波动大、可能为负,且锁在结汇额度后面(每人每年 5 万美元,月均约 3 万元),**不能用来按月偿付人民币负债**。工具把它与"锁定息差"分两行显示,并在资产负债表里用斜纹单独标注境外权益。把权益溢价当成"必赚的息差"正是杠杆交易里最贵的认知错误——它只能进 α,不能进现金流下限。

判定逻辑也据此设定:**DSCR ≥ 1 且固收腿能覆盖生活,权益收益才是纯加分项**;一旦要靠权益腿补生存现金流,就触发"依赖 α / 偿付越线"。

## 六、计算口径(可审计)

- **月供**:等额本息,公式与 Excel `PMT` 完全一致。锚点 `PMT(3.3%/12, 270, 1760452.12) = 9246.2109633523`,本引擎输出 13 位有效数字一致;中途不取整。
- **资产负债表**:净资产 = 总资产 − 负债;两柱等高即恒等式成立。房产层杠杆 = 1/首付比(默认 5×);家庭层杠杆 = 总资产/净资产(默认 ~1.5×)——刻意区分,避免把"房产层 5 倍"误当成"家庭层"。
- **WACC(加权债务成本)** = (公积金额×公积金利率 + 商贷额×商贷利率) / 总贷款。严格说只含债务侧,故标注"= WACC(仅债务侧)"。
- **DSCR** = 妻子税后月收入 / 月供;**全口径覆盖** = 收入 /(月供 + 生活)。判定:全口径 ≥1 且月结余 ≥3000 为结构稳健;月供 > 收入为偿付越线。
- **息差**:锁定息差 = r* − WACC;权益溢价 = 美股 E[R] − WACC(单列、加警示)。**NPV(A−B) = 商贷金额 − PV(商贷月供流 @ r*/12)**,盈亏临界点恰为商贷利率。
- **看跌"保费"** = 安全垫 ×(r* − 无风险利率);保护深度 = 安全垫 ÷(月供 + 生活)。
- **久期匹配**为演示开关:30 年按揭=匹配(绿),经营贷≈3 年=错配(红)。
- **契税**:首套 ≤140㎡ 1%,>140㎡ 1.5%。

**自检**:`cd test && node page_test.js`,30 项断言(恒等式、各比率、息差拆分、久期切换、看跌保费、判定阈值、契税、非法输入、两方式切换、Excel 锚点等)应全部 PASS。
